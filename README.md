# Classificador de emails 

Aplicação web que utiliza Inteligência Artificial para classificar emails e sugerir respostas automáticas, ajudando a equipe a economizar tempo com mensagens que não precisam de atenção imediata.

- **Produtivo:** emails que precisam de ação ou resposta.  
- **Improdutivo:** emails que não precisam de resposta imediata.

---

## Como usar

1. Acesse a aplicação pelo link do deploy.  
2. Cole o texto do email na caixa de texto.  
3. Clique em **Analisar email**.  
4. O resultado aparecerá logo abaixo, mostrando:
   - A categoria do email (Produtivo ou Improdutivo)  
   - Uma sugestão de resposta automática  
---

## Tecnologias

- Python + FastAPI (backend)  
- HTML + CSS (frontend)  
- Transformers da Hugging Face para classificação de texto
